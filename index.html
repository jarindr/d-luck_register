คุณสมบัติของผู้สมัคร

คุณสมบัติทั่วไปของผู้สมัคร

1. ชาย-หญิง มีอายุระหว่าง 18-35 ปี

2. มีความสามารถพิเศษเฉพาะตัวด้านใดด้านหนึ่งดังนี้

- การเต้น ไทย สากล ประยุกต์

- Gymnastics

- Aerial Acrobat

- Stuntman

- ศิลปะการต่อสู้ทุกแขนง เช่น มวยไทย-มวยสากล / กระบี่ กระบอง

- ทักษะการแสดงโขน ( เน้นการตีลังกา และความแข็งแรงของร่างกาย )

- โชว์ควงกระบองไฟ

- หรือความสามารถอื่นๆ

3. มีร่างกายแข็งแรง ไม่มีโรคประจำตัวร้ายแรง หรือ โรคที่เป็นอุปสรรคต่อการแสดง เช่น โรคหัวใจ โรคลมชัก โรคหอบหืด โรคเบาหวาน เป็นต้น

4. สามารถไปทำงานที่พัทยา จ.ชลบุรี ระยะยาวได้

5. สามารถซ้อมและทำการแสดงได้ตามคิวที่ทีมงานกำหนด (เช็ครายละเอียดได้ในวันที่มาสัมภาษณ์และทดสอบความสามารถ)

คุณสมบัติพิเศษของผู้สมัครตัวละครพิเศษ ( คนแคระ )

1. เพศชายที่มีอายุ 18-30 ปี สูงไม่เกิน 140 เซนติเมตร มีร่างกายแข็งแรง

2. มีความสามารถในการแสดง หรือ การเต้น จะได้รับการพิจารณาพิเศษ

3. ต้องไม่มีโรคประจำตัวตามที่ระบุข้างต้น

การเตรียมตัวในวันสัมภาษณ์และทดสอบความสามารถ

1. การแต่งกายควรเป็นชุดกีฬาหรือชุดที่สามารถเคลื่อนไหวสะดวก

2. รองเท้าควรเป็นรองเท้าผ้าใบหรือรองเท้าเฉพาะทางสำหรับการเต้นหรือโชว์ความสามารถ ( ผู้สมัครต้องเตรียมมาเอง )

3. ต้องเตรียมอุปกรณ์ประกอบการโชว์ความสามารถพิเศษมาเอง (ถ้ามี) เช่น กระบี่ กระบอง เป็นต้น

4. ในการคัดเลือกต้องคำนึงถึงความปลอดภัยเป็นหลัก เพราะหากเกิดการบาดเจ็บอันมาจากการแสดงความสามารถของท่านในขณะคัดเลือก ทีมงานขอไม่รับผิดชอบใดๆ


*แจ้งให้ทราบเพื่อเป็นข้อมูลประกอบการตัดสินใจในการเข้าร่วมแสดง
